# FLUX Assessment

Web app per assessment psicologico degli atleti.

## Deploy su Vercel
1. Carica questo codice su GitHub.
2. Crea nuovo progetto su Vercel importando il repository.
3. Configura le variabili d'ambiente:
   - `VITE_SUPABASE_URL`
   - `VITE_SUPABASE_ANON_KEY`
   - `VITE_ADMIN_PASSWORD`

## Sviluppo Locale
```bash
npm install
npm run dev
```
